A Solução

Gang of Four (GoF)

Criacionais

Abstraem a instanciação. O sistema torna-se independente de como seus objetos são criados.

→ Factory Method → Builder

Estruturais

Preocupam-se com a composição de classes para formar estruturas complexas.

→ Facade → Decorator

Comportamentais

Atribuição de responsabilidades e algoritmos de comunicação entre os objetos.

→ Strategy → State → Observer

NotificationStrategy.php (Depois)

// 1. O Contrato de Estratégia
interface NotificationStrategy {
    public function send(User $user, string $message): void;
}

// 2. ✅ O Serviço recebe a Estratégia via Injeção de Dependência (Polimorfismo)
class UserRegistrationService {
    public function __construct(private NotificationStrategy $notifier) {}
    
    public function register(User $user) {
        $this->notifier->send($user, 'Bem vindo ao SyncMatch!'); 
    }
}
